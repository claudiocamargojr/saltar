Para usar, deve-se passar como argumento o diretorio completo da pasta "Analyses" do cuckoo. Ex:

python rename_pcaps.py /home/martell/cuckoo/Analyses

Este script renomeia os arquivos PCAP de dump.pcap para <hash malware>.pcap. Não altera o arquivo pcap de diretório, apenas renomeando o arquivo em seu local de origem.
